# Portfolio — Talvin Ackbaraly

Portfolio personnel minimaliste dédié à la présentation des projets et compétences en vision par ordinateur, IA et calcul GPU.

## Vue d'ensemble

- **Stack** : HTML5, CSS3 natif et JavaScript ES Modules, sans framework ni dépendances externes.
- **Internationalisation** : Support bilingue (Français / Anglais) commutable dynamiquement.
- **Architecture de données** : Le contenu (parcours, projets, compétences) est entièrement séparé de la présentation et centralisé dans `js/data.js`.
- **Déploiement** : Hébergement sur Vercel avec intégration continue depuis le dépôt GitHub `talvinckb/portfolio`.

## Arborescence

```
.
├── index.html          # Structure sémantique et accessibilité
├── css/
│   └── style.css       # Design system, thèmes sombre/clair et adaptabilité mobile
├── js/
│   ├── data.js         # Source de données bilingue
│   └── main.js         # Logique d'affichage et d'interaction
└── assets/
    └── favicon.svg     # Identité visuelle (favicons)
```
